# Psychological Schema Assessment MVP

This is a Streamlit app for assessing psychological schemas based on a 100-question survey. It calculates scores for 20 core schemas, identifies top schemas, and provides action plans.

## Setup

1. Clone the repo.
2. Install dependencies: `pip install -r requirements.txt`
3. Run locally: `streamlit run app.py`
